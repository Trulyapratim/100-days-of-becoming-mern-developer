JavaScript is a Dynamically typed language. Meaning their exists data types. But variables are not bound to it. 

eg: - let message = 'hello world'; // here it is a string
      message = 12232;  // now it become a number and its completely fine.

Their are 8 basic Data Types in Js : -

1. number : for numbers,  int, float etc.
2: bigint : for very big number and we put n at back of the number to show   it is an big int. eg: 820585852955325523534535n;
3: string :  for strings 
4: boolean : for "true" and "false"
5: null :    for unknown values – a standalone type that has a single value null
6. undefined : for unassigned variables
7: symbol : for unique identifiers
8: object : for complex data structures

Now let talk about an special operator : - typeof x; 

this is a special operator that tell the data type of any variable and expression.
